# Cualli, LLC - Official Website Repository

Welcome to the official repository for the Cualli website. Cualli is a forward-thinking biotechnology company dedicated to developing science-based, highly effective probiotics. Our flagship product is derived from the award-winning 2025 UNC iGEM project, utilizing an engineered *Escherichia coli* Nissle 1917 (EcN) strain to sequester PFAS ("forever chemicals") in the human gut.

## 🧬 The Science
Our technology focuses on disrupting the TolC-AcrB efflux system within the EcN bacteria. By creating a stable AcrB knockout via recA-mediated recombination, we maximize the intracellular retention of perfluoroalkyl substances (like PFOA). This effectively captures the toxins before they can be absorbed into the human bloodstream.

## 🎨 Design Philosophy
The UI/UX of this website is heavily inspired by Apple and Google's design systems:
- **Glassmorphism:** Navigation and data cards feature backdrop blurring for a layered, modern feel.
- **Typography:** System fonts with tight kerning and large, bold headers ensure readability and impact.
- **Dark Mode First:** A deep, immersive dark theme that perfectly complements the Cualli logo, accented with bio-luminescent cyan and emerald gradients to represent our engineered biological solutions.

## 📂 File Structure

- `science.html`: A deep dive into the synthetic biology and mechanism of action behind our EcN probiotic.
- `styles.css`: The global stylesheet containing our design system.
- `image_6b4127.png`: The Cualli logo.

Simply clone this repository and open `index.html` in any modern web browser. No build steps or package managers are required for this static, high-performance site.
